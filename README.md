# CodeAlpha - CGPA Calculator (C++)

## 📌 Overview

This project is a simple CGPA Calculator developed in C++ as part of the CodeAlpha C++ Programming Internship.

The program accepts grades and credits for multiple courses, converts letter grades into grade points, and calculates the final CGPA.

---

## ✨ Features

* Calculates CGPA for multiple courses
* Uses vectors to store grades and credits
* Converts letter grades into grade points
* Displays entered course details
* Handles invalid grade input

---

## 🛠 Technologies Used

* C++
* Standard Template Library (STL)
* Vectors
* Conditional Statements
* Loops

## ▶️ How to Run

Compile:

```bash
g++ main.cpp -o cgpa
```

Run:

```bash
./cgpa
```
